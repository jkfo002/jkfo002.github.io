# readthedocs

# 配置总结

  • 注册账号 (略)

  • 本地构建

    基于 python 3.12

    一路回车，可选项见参考

    更改主题（可选）

    md 支持（可选）

    配置 source/conf.py

  • github (略)

  • 项目托管

    若需要项目托管的话，需要在 github 中添加 .readthedocs.yaml

    requirements.txt 为必须

    sphinx_rtd_theme
    recommonmark

  • 更新

    本地更改后 push 到 github 就可以，readthedocs 自动更新
